Word Scamble

by Faraaz Kurawle

The foundation for the design of this website was laid while I was working on my personal project MineSweeper, developed using Python and the Tkinter framework.

Later, this design was adapted into a Word Scramble game during a competition called TekZone: GameForge, where my task was to create a word scramble game. Instead of building a dataset from scratch, I thought—why not use APIs to fetch random words and their definitions? And that’s how the core functionality of my Word Scramble game came to life.
